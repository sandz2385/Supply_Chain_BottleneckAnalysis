# 📊 Sales Analysis Dashboard  

This project analyzes sales data using SQL and Power BI to identify key business insights across regions, product categories, and customer trends.

## Contents:
- Interactive dashboard showing KPIs and trends  
- Dataset used for analysis  
- Executive summary of insights  

## Key Insights:
- Certain regions contribute more significantly to total sales  
- Sports and fitness categories show strong performance  
- Customer ratings remain stable across segments  
- Sales trends vary across locations and time  

## Tools Used:
- SQL (data extraction and transformation)  
- Power BI (data visualization)  
- Data analysis techniques
